# Get you some Healthy Potatoes!

## MERN (MongoDB, ExpressJS, ReactJS, and NodeJS) Stack App.

Current working site: https://healthypotatoes.herokuapp.com/

To install 
````
npm i
````

To Start/run
````
npm start
````
